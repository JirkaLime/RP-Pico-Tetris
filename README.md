### Co to umí?
#### Program umí plně pohyovat s Tetrominy [1](https://en.wikipedia.org/wiki/Tetromino), umí s nimi rotovat bere v potaz kolize.
#### Je 7 barevných provedení, 7 růných tvarů a každý z nich může rotovat.
